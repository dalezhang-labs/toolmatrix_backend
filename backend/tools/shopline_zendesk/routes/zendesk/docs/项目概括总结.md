# Shopline by OmnigaTech 项目概括总结

## 项目概述

Shopline by OmnigaTech 是一个完整的 SaaS 电商解决方案，为 Zendesk 提供深度集成的电商管理功能。该项目采用前后端分离架构，提供多租户支持、订阅管理、客户管理、订单管理等核心电商功能。

## 技术架构

### 后端架构 (FastAPI)

#### 技术栈
- **框架**: FastAPI (v0.115.5) - 高性能异步 Python Web 框架
- **数据库**: PostgreSQL + SQLAlchemy (异步 ORM)
- **认证**: JWT + OAuth 2.0
- **支付**: Stripe API (v11.3.0)
- **部署**: Render Cloud Platform
- **Python版本**: 3.13 兼容

#### 核心组件
1. **多层架构设计**
   - `main.py`: 应用入口，配置中间件和路由
   - `app/routers/`: API 路由层，处理 HTTP 请求
   - `app/services/`: 业务逻辑层，处理核心业务
   - `app/models/`: 数据模型层，定义数据结构
   - `app/middleware/`: 中间件层，处理认证和租户隔离

2. **中间件系统**
   - `AuthMiddleware`: 处理 Zendesk 令牌认证
   - `TenantMiddleware`: 实现多租户隔离
   - CORS 中间件: 跨域请求支持
   - TrustedHost 中间件: 主机安全验证

### 前端架构 (Next.js)

#### 技术栈
- **框架**: Next.js 14.2.5 + React 18.3
- **UI框架**: 自定义组件系统
- **认证**: NextAuth.js v4 + Google OAuth
- **支付集成**: Stripe.js
- **类型系统**: TypeScript 5.4
- **测试**: Playwright

#### 核心功能模块
1. **页面结构**
   - 首页营销页面
   - 用户认证系统（登录/注册/邮箱验证）
   - 订阅管理界面
   - 账户设置页面
   - 管理员控制台

2. **API 路由** (`pages/api/`)
   - 认证接口: Google OAuth、邮箱验证码、会话管理
   - Stripe 集成: 支付会话、订阅管理、Webhook 处理
   - 用户管理: 账户信息、权限控制

## 数据库架构

### 核心数据表

1. **租户管理**
   - `tenants`: 租户信息表（Zendesk 子域名映射）
   - `user_tenants`: 用户-租户关联表（多对多）

2. **用户系统**
   - `site_users`: 网站用户（支持 Google OAuth）
   - `users`: 系统用户（已废弃，迁移到 site_users）

3. **订阅管理**
   - `subscriptions`: 内部订阅记录
   - `user_stripe_subscriptions`: Stripe 订阅详情

4. **系统监控**
   - `api_logs`: API 调用日志
   - `cache_data`: 缓存数据

### 数据关系
- 一个用户可管理多个 Zendesk 租户
- 每个租户对应一个 Shopline 商店
- 订阅与用户直接关联，支持多级计划

## API 端点结构

### 客户管理 (`/api/customers`)
- 搜索客户（邮箱、电话、ID）
- 客户详情查询
- 客户订单历史

### 订单管理 (`/api/orders`)
- 订单列表和详情
- 订单状态更新
- 订单取消和退款
- 客户订单查询

### 物流管理 (`/api/logistics`)
- 物流信息查询
- 包裹跟踪
- 配送时间线

### 订阅管理 (`/api/subscriptions`)
- 订阅计划查询
- 创建和取消订阅
- 当前订阅状态

### Stripe 集成 (`/api/stripe`)
- 支付会话创建
- 客户门户访问
- Webhook 事件处理

### 租户管理 (`/api/tenants`)
- 租户创建和更新
- 通过子域名查询
- 租户配置管理

## 集成服务

### 1. Shopline API 集成
- **用途**: 连接 Shopline 电商平台
- **功能**: 订单、客户、物流数据同步
- **认证**: JWT Bearer Token
- **API版本**: v20250601

### 2. Stripe 支付集成
- **订阅计划**: Basic、Professional、Enterprise
- **功能**: 
  - 订阅创建和管理
  - 支付处理
  - 客户门户
  - Webhook 事件处理

### 3. Zendesk 集成
- **集成方式**: Zendesk App Framework (ZAF)
- **功能**: 
  - 客户信息展示
  - 订单历史查询
  - 物流跟踪
  - 多租户隔离

### 4. Google OAuth 2.0
- **用途**: 用户快速登录
- **实现**: NextAuth.js 集成
- **数据存储**: google_id 关联到 site_users

## 安全机制

### 认证与授权
1. **多层认证**
   - JWT 令牌认证（后端 API）
   - NextAuth 会话管理（前端）
   - Google OAuth 2.0（第三方登录）

2. **安全措施**
   - 密码使用 bcrypt 加密
   - 敏感信息环境变量存储
   - HTTPS 强制使用
   - CORS 策略配置

### 多租户隔离
- 基于 Zendesk 子域名的租户识别
- 请求级别的租户上下文注入
- 数据库级别的租户数据隔离

## 部署架构
